## 🤝 Contribuição

Contribuições são sempre bem-vindas! Se você deseja colaborar com o desenvolvimento do **PrismaOfertas**, siga os passos abaixo:

1. Faça um **Fork** deste repositório.
2. Crie uma nova branch para sua funcionalidade ou correção:

   ```bash
   git checkout -b feature/minha-funcionalidade
   ```
3. Realize as alterações necessárias e faça seus commits:

   ```bash
   git commit -m "feat: adiciona nova funcionalidade"
   ```
4. Envie suas alterações para o seu repositório:

   ```bash
   git push origin feature/minha-funcionalidade
   ```
5. Abra um **Pull Request** descrevendo detalhadamente as mudanças propostas.

### Boas práticas

* Mantenha o código organizado e bem documentado.
* Siga o padrão de estrutura e nomenclatura utilizado no projeto.
* Escreva mensagens de commit claras e descritivas.
* Teste suas alterações antes de enviar um Pull Request.
* Sempre verifique se não há conflitos com a branch principal.

Toda contribuição que ajude a melhorar a qualidade, o desempenho, a segurança ou a experiência dos usuários do **PrismaOfertas** será muito bem-vinda. 🚀
